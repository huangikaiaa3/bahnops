# OfferOps Roadmap

## Core idea

Build one project that grows with the learning plan instead of making a separate toy app for every topic.

## Current scope

1. Parse and store job-search data.
2. Expose it through an API.
3. Add a RAG assistant for jobs, resume notes, and recruiter conversations.
4. Layer in analytics and event-driven pieces later.

## Non-goals right now

- production deployment
- cloud complexity
- choosing every tool upfront
